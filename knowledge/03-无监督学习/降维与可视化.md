# 降维与可视化

## 概念介绍

除了PCA之外，还有许多专门用于可视化的降维算法。数据可视化是理解高维数据最直观的方式——把几百维的数据投射到2D平面上，用颜色区分类别，一眼就能看出数据的结构。最常用的可视化降维算法有t-SNE和UMAP，它们能发现PCA无法捕捉的非线性结构。

## 核心原理

### t-SNE（t-distributed Stochastic Neighbor Embedding）

t-SNE的核心思想：在高维空间中相似的点，在低维空间中也应该靠近。

1. 在高维空间中，计算每对点之间的相似度（用高斯分布衡量）
2. 在低维空间中，用t分布来表示点之间的相似度
3. 通过优化，使低维空间的相似度分布尽可能接近高维空间

t-SNE的优势是能保留局部结构（相似的点聚在一起），但计算慢，不适合大数据。

### UMAP（Uniform Manifold Approximation and Projection）

UMAP是t-SNE的升级版，速度更快，能保留更多全局结构，且支持增量学习。近年来越来越流行。

### 降维方法对比

- **PCA**：线性，速度快，保留全局结构，适合预处理
- **t-SNE**：非线性，速度慢，保留局部结构，适合可视化
- **UMAP**：非线性，速度快，保留局部+部分全局结构，适合可视化和预处理

## 代码实现

```python
from sklearn.manifold import TSNE
from sklearn.decomposition import PCA
from sklearn.datasets import load_digits
from sklearn.preprocessing import StandardScaler
import matplotlib.pyplot as plt
import numpy as np

# 加载手写数字数据集（64维）
digits = load_digits()
X = digits.data   # 64维
y = digits.target  # 0-9数字

# 标准化
X_scaled = StandardScaler().fit_transform(X)

# ========== PCA降维 ==========
X_pca = PCA(n_components=2).fit_transform(X_scaled)

# ========== t-SNE降维 ==========
X_tsne = TSNE(n_components=2, perplexity=30, random_state=42, n_iter=1000).fit_transform(X_scaled)

# ========== 对比可视化 ==========
fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(16, 6))

# PCA图
scatter1 = ax1.scatter(X_pca[:, 0], X_pca[:, 1], c=y, cmap='tab10', s=10, alpha=0.7)
ax1.set_title('PCA降维')
ax1.set_xlabel('PC1')
ax1.set_ylabel('PC2')
plt.colorbar(scatter1, ax=ax1)

# t-SNE图
scatter2 = ax2.scatter(X_tsne[:, 0], X_tsne[:, 1], c=y, cmap='tab10', s=10, alpha=0.7)
ax2.set_title('t-SNE降维')
ax2.set_xlabel('t-SNE1')
ax2.set_ylabel('t-SNE2')
plt.colorbar(scatter2, ax=ax2)

plt.tight_layout()
plt.savefig("dim_reduction_compare.png", dpi=100)
plt.show()

# t-SNE通常能看到更清晰的类别分离
print("t-SNE完成，可以看到不同数字被清晰地分开了")
```

```python
# ========== UMAP（需要安装：pip install umap-learn） ==========

```

## 适用场景

- 探索性数据分析（看看数据长什么样）
- 高维数据可视化（100+维降到2D）
- 验证聚类/分类效果（降维后看类别是否可分）
- 论文/报告中的数据展示

## 常见易错点

1. **把t-SNE当特征提取**：t-SNE每次运行结果不同，不能用于模型训练的特征
2. **t-SNE的perplexity不调**：perplexity控制邻域大小，通常5-50，需要尝试
3. **过度解读t-SNE图**：t-SNE的簇大小和距离不代表真实的密度和距离
4. **大数据集直接用t-SNE**：先用PCA降到50维，再用t-SNE降到2维

   - 答案：大

3. **简答题**：为什么t-SNE不适合作为模型训练的特征提取方法？
   - 答案：t-SNE每次运行结果不同（非确定性）、不能对新数据做变换（只拟合已有数据）、计算复杂度高。
