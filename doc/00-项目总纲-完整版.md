## 3.3 编排器设计（Orchestrator）

### 意图识别规则

| 用户意图关键词 | 调度Agent |
|--------------|----------|
| 首次对话/介绍自己/学习背景 | ProfileAgent |
| 讲解/解释/什么是/原理 | DocAgent |
| 思维导图/知识结构/梳理 | MindMapAgent |
| 练习/题目/测试/做题 | QuizAgent |
| 推荐/阅读/论文/资料 | ReadingAgent |
| 代码/实操/案例/项目 | CodeCaseAgent |
| 学习路径/规划/计划/怎么学 | PathAgent |
| 问题/疑问/为什么/怎么理解 | TutorAgent |

### 协作流程示例

**流程1：新用户首次使用**
```
用户："我是大二计算机专业，线代基础还行，想学机器学习做推荐系统"

Step1: Orchestrator → ProfileAgent
  → 生成初始画像（知识基础：线代中等/编程待确认/ML零基础）

Step2: 画像触发 → PathAgent
  → 生成个性化学习路径

Step3: 路径第一个节点 → 并行触发5个资源Agent
  → DocAgent("基础概念", 画像) → 课程文档
  → MindMapAgent("基础概念") → 思维导图
  → QuizAgent("基础概念", "入门") → 练习题
  → ReadingAgent("基础概念", 画像) → 推荐阅读
  → CodeCaseAgent("基础概念", 画像) → 代码案例

Step4: 资源打包 → 推送到前端资源中心
```

**流程2：用户学习过程中提问**
```
用户："决策树的剪枝策略我不太理解"

Step1: Orchestrator → TutorAgent
  → 查询知识库 → 生成解答（文字+图解+代码）

Step2: 同时 → ProfileAgent（更新易错点维度）
  → weak_points += "决策树剪枝"

Step3: 画像变化 → PathAgent（动态调整路径）
  → 在当前节点增加补充资源
```

## 3.4 Agent核心代码设计

```java
// Agent抽象基类
public abstract class AgentBase {
    protected String name;
    protected String role;
    protected String systemPrompt;

    public abstract AgentMessage process(AgentMessage input);
}

// 编排器
@Component
public class Orchestrator {
    @Autowired
    private Map<String, AgentBase> agentMap;  // Spring自动注入所有Agent

    public AgentMessage dispatch(String intent, AgentMessage input) {
        AgentBase agent = routeByIntent(intent);
        return agent.process(input);
    }
}
```

**设计亮点**：Spring Bean + 策略模式，每个Agent是独立Bean，编排器通过`@Autowired Map<String, AgentBase>`自动收集所有Agent，新增Agent只需加`@Component`注解即可注册。

```sql
CREATE TABLE student (
    id          BIGINT PRIMARY KEY AUTO_INCREMENT,

    avatar      VARCHAR(255) DEFAULT NULL,
    created_at  DATETIME DEFAULT CURRENT_TIMESTAMP,
```

### 学习画像表（6维度）

```sql
CREATE TABLE student_profile (
    id               BIGINT PRIMARY KEY AUTO_INCREMENT,
    student_id       BIGINT NOT NULL,
    knowledge_base   JSON COMMENT '知识基础',
    cognitive_style  JSON COMMENT '认知风格',
    learning_goal    JSON COMMENT '学习目标',
```

```sql
CREATE TABLE chat_message (
```

### 学习资源表

```sql
CREATE TABLE learning_resource (

    title            VARCHAR(200) NOT NULL,
    content          JSON NOT NULL,

    difficulty       VARCHAR(20) DEFAULT 'medium',

    INDEX idx_student_id (student_id),
```
