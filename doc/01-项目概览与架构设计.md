## 赛题目标

借助大模型技术体系，融合前沿AI技术，构建高等教育个性化学习资源体系，开发智能学习智能体系统，满足学生的个性化、多模态学习需求。以**机器学习**课程为切入点，实现个性化资源的自动化生成与建设。

| # | 资源类型 | 生成方式 | 负责 |

## 6维度学习画像定义

| 维度 | 说明 | 机器学习场景问法 |
|------|------|----------------|
| 知识基础 | 当前掌握的知识点和水平 | "学过高数/线代/概率论/Python吗？" |
| 认知风格 | 视觉型/听觉型/实践型偏好 | "喜欢公式推导/图表/动手写代码？" |
| 学习目标 | 短期/长期目标 | "考研/就业/科研/兴趣？想做什么方向？" |
| 易错点偏好 | 常犯错误类型 | "之前在哪些概念上卡过？" |
| 学习节奏 | 快/中/慢，每日投入时间 | "每天能学多久？希望多快学完？" |
| 兴趣方向 | AI细分领域偏好 | "对CV/NLP/推荐系统/数据分析哪个感兴趣？" |

## 非功能性要求对照

| 赛题要求 | 实现方案 |
|---------|---------|
| 界面美观、流式输出、Markdown渲染、卡片化展示 | React + Ant Design + react-markdown + 流式渲染 |

## 风险控制

| 风险 | 对策 |
|------|------|
| 讯飞API响应慢 | 流式输出 + 进度条 + Prompt精简减token |
| 3天做不完8个Agent | Agent模板化开发，差异只在System Prompt，每个最多2h |
| 生成内容质量差 | Day4专门优化Prompt + 知识库提供高质量参考 |
| 前后端联调阻塞 | D1-D2各自Mock开发，D3集中联调 |
| 知识库不完整 | 第三位前2天全力保障，是整个系统的"弹药" |
