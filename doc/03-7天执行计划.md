# 7天执行计划

## 总体节奏

```
第一阶段（Day1-3）：MVP开发 —— 三人全力编码，目标：功能全跑通
第二阶段（Day4-7）：打磨提交 —— 优化+文档+视频，目标：品质达标
```

### 三人分工总览

| 角色 | 第一阶段（Day1-3） | 第二阶段（Day4-7） |
|------|-------------------|-------------------|
| **后端（Java）** | 框架搭建 + 8个Agent + API + 联调 | Prompt优化 + 性能稳定 + 部署 |
| **前端（React）** | 6个页面 + 各类组件 + 联调 | UI美化 + 交互细节 + 兼容性 |
| **第三位（文档&视频）** | 知识库素材 + 文档框架 | 文档终稿 + PPT + 演示视频 |

## 第一阶段：MVP开发

### Day 1 —— 基础搭建 + 对话 + 画像

#### 后端任务

| 时段 | 任务 | 产出物 | 预计耗时 |
|------|------|--------|---------|
| 上午 | Spring Boot脚手架搭建（Maven项目 + 包结构 + 配置） | 项目可启动 | 2h |
| 上午 | MySQL建表（6张核心表）+ MyBatis-Plus配置 | 数据库就绪 | 1h |

| 下午 | **AgentBase**抽象类 + **Orchestrator**编排器骨架 | 框架v0.1 | 1h |
| 晚上 | **ProfileAgent**实现（6维度画像抽取） | 画像Agent可用 | 1.5h |
| 晚上 | **ChatController**（WebSocket STOMP） + 画像CRUD API | 接口就绪 | 2h |
| 晚上 | 用Postman联调：对话 → 画像生成 → 画像查询 | 后端闭环 | 1h |

#### 前端任务

| 时段 | 任务 | 产出物 | 预计耗时 |
|------|------|--------|---------|
| 上午 | Vite + React + TypeScript脚手架 + Ant Design + 路由 | 项目可运行 | 2h |
| 上午 | 整体Layout（侧边栏导航 + 主内容区） | 页面框架 | 1h |
| 下午 | **对话页**：聊天列表 + 消息输入 + 流式输出渲染 | 聊天UI | 2h |
| 下午 | **Markdown渲染**：react-markdown + 代码高亮 + 公式 | 渲染能力 | 1.5h |
| 晚上 | **画像页**：6维雷达图(ECharts) + 画像详情卡片 | 画像UI | 1.5h |
| 晚上 | WebSocket连接 + 对接对话API + 对接画像API | 前后端联通 | 2h |

#### 第三位任务

| 时段 | 任务 |
|------|------|
| 全天 | ① 整理机器学习课程知识库素材（6大模块Markdown文档） |
